Apache Config

¡Bienvenido a la Configuración de Apache en Sistemas Operativos!

Esta página es una guía detallada para configurar Apache en varios sistemas operativos usando Docker y Docker Compose.

Aprenderás a configurar Apache para servir múltiples sitios web, a gestionar módulos y a personalizar la configuración según tus necesidades.

Leer más

Configuración Básica

Para comenzar, necesitas tener Docker y Docker Compose instalados en tu sistema. Luego, puedes usar los archivos de configuración proporcionados para construir tu contenedor de Apache.


# Ejemplo de Dockerfile básico
FROM httpd:2.4
COPY ./config/my-httpd.conf /usr/local/apache2/conf/httpd.conf
COPY ./www/ /usr/local/apache2/htdocs/

Configuración Avanzada

Para una configuración más avanzada, puedes personalizar los archivos de configuración de Apache, como httpd.conf y httpd-vhosts.conf. También puedes agregar módulos adicionales y scripts de inicio personalizados.


# Ejemplo de Dockerfile avanzado
FROM httpd:2.4
RUN apt-get update && apt-get install -y vim
COPY ./config/ /usr/local/apache2/conf/
COPY ./scripts/startup.sh /usr/local/bin/startup.sh
RUN chmod +x /usr/local/bin/startup.sh
